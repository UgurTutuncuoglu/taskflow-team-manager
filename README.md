# TaskFlow - Team Task Manager

## Project Description

TaskFlow is a React-based Team Task Management application developed for CRUD operations practice.

The application allows users to:

* Add tasks
* List tasks
* Update tasks
* Delete tasks

All data is stored using browser localStorage.

---

# Technologies Used

* React
* JavaScript
* Bootstrap
* CSS
* localStorage

---

# Features

* Create Task
* Read Task List
* Update Task
* Delete Task
* Status Management
* Date Validation
* Estimated Hours Validation
* Responsive Design

---

# Task Fields

Each task contains:

* Task Name
* Assigned Person
* Start Date
* End Date
* Estimated Hours
* Status
* Description

---

# Status Types

* Unresolved
* In Progress
* Team Discussion
* Resolved
* Closed

---

# Validation Rules

* Empty fields are not allowed
* Start Date cannot be later than End Date
* Estimated Hours must be a positive integer
* Estimated Hours cannot exceed 1000
